Module 03

Final Project

This page showcases the final HCI project, including concept, interaction, and demo highlights. Click each section header to expand or collapse the details.

Our project is “An Intelligent Study Material Management System”, designed to help students efficiently organize and retrieve scattered course materials. Target users are senior undergraduates and postgraduates handling multiple courses. The system addresses problems like finding specific knowledge points, avoiding manual organization, and preserving privacy through local AI processing.

The system features three main interaction paths:

  • Ingestion: Drag & drop files → Process & Categorize → AI auto-tags and updates file cards.
  • Browse & Manage: Scan tag cloud → browse file cards → read AI summaries → add manual tags.
  • Semantic Query: Enter natural-language questions → system retrieves documents → RAG generates answers in chat window.

These paths form a simple loop: upload → explore → ask, emphasizing recognition over recall.

Key features demonstrated in the prototype include:

  • Tag Cloud Visualization: overview of knowledge coverage.
  • Natural Language Semantic Query (RAG): direct answers without remembering filenames.
  • Manual Tag Supplementation: users can customize organization alongside AI tags.
  • Content Layering: Tag cloud → File cards → Chat query for detailed answers.
  • Local AI Models: preserves privacy and supports reproducible retrieval.

Evaluation showed high usability, low learning cost, and identified actionable improvements like delete confirmation.
