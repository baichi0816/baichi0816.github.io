# Jintian YAN | HCI Personal Portfolio

This is a static personal portfolio website for an HCI course project diary.

## Final website URL

After deployment, the website should be available at:

```text
https://baichi0816.github.io
```

## GitHub repository name

Create a public GitHub repository named:

```text
baichi0816.github.io
```

## Deployment steps

1. Log in to GitHub.
2. Create a new public repository named `baichi0816.github.io`.
3. Upload all files in this folder:
   - `index.html`
   - `style.css`
   - `README.md`
4. Go to `Settings` → `Pages`.
5. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Click `Save`.
7. Wait for GitHub Pages to deploy.

## Edit content

To change the homepage text, edit `index.html`.

To change colors/layout, edit `style.css`.

## Notes

This site uses an industrial functional visual style inspired by dark UI, terminal interfaces, grid systems, and warning-stripe motifs. It does not use copyrighted game assets.
